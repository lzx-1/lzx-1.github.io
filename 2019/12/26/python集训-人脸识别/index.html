python集训 _人脸识别

最近集训了为期四天的python培训，主要是学习人脸识别。
首先说一下遇到的问题汇总，1.python的环境配置 2.python的pip安装 3.pycharm的使用
关于python的环境配置，目前有很多种版本的python。如3.7.0，3.8.0，我之前安装的是3.8的python，所以有很多老师提供3.7版本pip我安装不上，pypi上也没有对应的3.8版本。所以我卸载了原先的3.8安装了3.7(最简单的卸载方法就是找到python的源文件双击uninstall，找不到就更新一个更高的版本安装后再卸载)。要记好python的安装路径。
然后是pip的安装，首先要确定你所需要的pip是否能和你的python版本对应的上，不对应可以去pypi的官网或者国内的镜像网站清华源找。找到之后下载到任意路径，然后打开下载到的文件夹，在上方输入cmd。输入pip install +文件名字(打出几个字母后按tab键)。输入pip list可以查看你所安装的全部pip，输入pip uninstall+文件名 可以卸载对应pip。
pycharm的使用，这才是重点。原来以前我写的那些代码都是用虚拟环境运行的，难怪import pygame一直失败。使用pycharm时如果不使用已安装的python环境他会给你创建一个虚拟环境，虚拟环境的优点是体积小，随项目移动。但是你安装的pip他是不会自动识别的，需要手动的添加进去。操作步骤为：文件-设置-项目-project Interpreter-右方的+号(-号就是删除)-然后搜索pip。
这次实训主要是对cv2的一些操作讲解以及认识到图片，声音，视频的数字化。图片就是一个三维的。可以使用print(cv2. imread(‘./文件名.’))查看文件是一个几维的。对图片来说，一二维确定图片的长宽，三维确定颜色(红绿蓝，全255为白，全0为黑)。贴一个代码 将人物的眼睛替换为心。
