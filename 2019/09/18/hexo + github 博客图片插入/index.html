hexo + github 博客图片插入

在写博客的时候 由于之前插件都已经配置好了
所以 只需要把我们写博客要用到的 图片放在对应文章下的文件夹里面
比如 ：要写一篇 test的文章
则source/_posts 会有对应的 test.md 和 test 文件夹
把需要用到的图片放置到 test文件夹中
然后在写文章的时候 在需要图片的位置添加一下语句
![](对应文章的文件夹/对应的图片 名称加后缀 )

this is a test

另外一个小提示:GitHub博客分类只需要在建立的md文件中第四行的tags行输入标签就好了。（记得在标签加一个空格）
